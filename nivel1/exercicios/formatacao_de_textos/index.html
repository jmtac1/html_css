Principais formatações:

Negrito / destaque

palavra em negrito usando a tag <b> (tag não semântica): palavra

palavra em destaque usando a tag <strong> (tag semântica): palavra

Itálico / ênfase

texto em Itálico usando a tag <i> (não semântica): palavra

texto em ênfase usando a tag <em> (semântica): palavra

Texto marcado

texto destacado usando a tag <mark>: palavra

texto destacado com cor diferente usando o parametro "style": palavra

Texto grande / pequeno

texto grande usando a tag <big>: palavra

texto pequeno usando a tag <small>: palavra

Texto excluído / sublinhado

texto excluído usando a tag <del>: palavra

texto sublinhado usando a tag <ins>: palavra

Texto sobrescrito / subscrito

texto sobrescrito usando a tag <sup>: palavra

texto subscrito usando a tag <sub>: palavra

Fonte mono espaçada

texto com fonte mono espaçada usando a tag <code>: palavra

Texto escrito com formato de código-fonte


num = input("digite um numero: ")
if num % 2 == 0:
    print("numero par")
else:
print("numero impar")

Citações simples / completas

citação simples usando a tag <q>: palavra

citação completa usando a tag <blockquote>:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Aperiam totam perspiciatis odit omnis asperiores in dolor corrupti esse sapiente. In aperiam illum modi ad sequi? Vero minus a ex quaerat.

Texto abreviado

indicar texto abreviado usando a tag <abbr>: plvr

Texto invertido

texto invertido usando a tag <bdo>: palavra
