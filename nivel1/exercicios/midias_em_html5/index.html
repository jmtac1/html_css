Como adicionar diferentes mídias em um site

imagens dinâmicas:

usando a tag <picture> e <source> podemos adicionar imagens de tamanhos diferentes, para que se adapte ao tamanho da tela

adicionar audios:

usando a tag <audio> e com o parametro "controls" podemos adicionar um player de audio

assim como as imagens podemos adicionar as tags <audio> e <source> para adicionar varios formatos de audio caso o navegador não consiga reproduzir algum formato especifico

adicionar videos hospedados localmente:

do mesmo jeito do audio, podemos adicionar um player de videos usando a tag <video> e o parametro "controls"

tambem do mesmo jeito do audio, podemos adicionar varios formatos de video em caso de não compatibilidade do navegador
