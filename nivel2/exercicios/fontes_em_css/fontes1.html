Tipos de fonte

serif, sans-serif, monospace, handwriting, display:

  • exemplo de fonte serif: palavra
  • exemplo de fonte sans-serif: palavra
  • exemplo de fonte monospace: palavra
  • não tem exemplo de fonte handwriting mas é literalmente o que o nome diz, simular uma escrita a mão
  • também não tem exemplo de fonte display, mas a fonte do filme jurassic park é um exemplo

é recomendado o uso das fontes do tipo "serif" para títulos

é recomendado o uso das fontes do tipo "sans-serif" para textos mais longos

é recomendado o uso das fontes do tipo "monospace" para código-fonte, pois os caracteres tem o mesmo tamanho

Tamanhos de fonte

medidas absolutas: cm, mm, in, px, pt, pc

medidas relativas: em, ex, rem, vw, vh, %

as medidas recomendadas para uso são: "px" ou "em"

1em é igual a 16px, que é a medida padrão nos navegadores

por isso é recomendado usar a medida "em", para poder trabalhar com as porcentagens da medida padrão

Peso e estilo da fonte:

algumas fontes podem ser alteradas a largura (weight) e o estilo (style) adicionando os parametros (font-weight) e (font-style)

Shorthand font:

todas essas configurações de fonte podem ser simplificadas com shorthand. Com a sintaxe para fazer essas configurações ficando assim por exemplo:

(font: italic bolder 1em Arial, Helvetica, sans-serif)

sendo a ordem a dos parametros a seguinte: (font-style), (font-weight), (font-size) e (font-family)
